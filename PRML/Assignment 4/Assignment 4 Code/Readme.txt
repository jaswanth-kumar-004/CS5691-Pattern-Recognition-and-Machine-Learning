Instructions to run the codes:

1)Synthetic data
* For speech recognition create a folder named "Synthetic_data"
* Download and save the train file ("train.txt") & test file ("dev.txt") within the "Synthetic_data" folder
* In terminal type command: python synthetic.py
* To go to next image after an image pops up while running the code, close the current image

2)Speech data
* For speech recognition create a folder named "Digits"
* Within the "Digits" folder create two subfolder named "dev" and "train"
* Download and save all the development data for all the classes (Team-34) of spoken digits into "dev" folder
* Download and save all the training data for all the classes (Team-34) of spoken digits into "train" folder
* In terminal type command: python digits.py
* To go to next image after an image pops up while running the code, close the current image

3)Handwritten data
* For speech recognition create a folder named "Written"
* Within the "Written" folder create two subfolder named "dev" and "train"
* Download and save all the development data for all the classes (Team-34) of handwritten characters into "dev" folder
* Download and save all the training data for all the classes (Team-34) of handwritten characters into "train" folder
* In terminal type command: python written.py
* To go to next image after an image pops up while running, close the current image

3)Image data
* For speech recognition create a folder named "Image"
* Within the "Image" folder create two subfolder named "dev" and "train"
* Download and save all the development data for all the classes (Team-34) of images into "dev" folder
* Download and save all the training data for all the classes (Team-34) of images into "train" folder
* In terminal type command: python image.py
* To go to next image after an image pops up while running, close the current image
